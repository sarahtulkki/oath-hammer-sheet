## Overview

The Oath Hammer Character Sheet is a full-featured character management tool designed specifically for the Oath Hammer tabletop RPG system. This extension provides players and Game Masters with a powerful, intuitive interface for tracking characters, managing inventory, and handling all aspects of gameplay within Owlbear Rodeo.

## Getting Started

### Installation

1. Open [Owlbear Rodeo](https://www.owlbear.rodeo/)
2. Navigate to the Extensions menu
3. Search for "Oath Hammer Character Sheet"
4. Click "Add" to install the extension

### Usage

1. **Launch the Extension**: Click the Oath Hammer icon in your Owlbear Rodeo toolbar
2. **Create or Load a Character**:
   - Click the **folder icon** to open the slot manager
   - Create a new character or load an existing one
3. **Fill Out Your Character**: Enter your character information - all changes save automatically
4. **Manage Your Characters**: Switch between up to 5 character slots as needed

## Character Management

### Creating Characters
- Use the slot manager (folder icon) to create new characters
- Each slot saves independently with its own last-modified timestamp
- Choose from 6 lineages and 10 classes

### Switching Characters
- Access the slot manager to view all your saved characters
- Switch between characters instantly
- Each character maintains its own complete state

### Importing & Exporting
- **Export**: Save your character as a JSON file to share with others or back up
- **Import**: Load characters from JSON files into any available slot

## Skills System

The character sheet includes all 26 Oath Hammer skills:

- Academics (Int)
- Acrobatics (Ag)
- Animal Handling (WP)
- Athletics (M)
- Brewing (Int)
- Carpentry (Ag)
- Defense (Ag / M)
- Dexterity (Ag)
- Diplomacy (WP)
- Discipline (WP)
- Fighting (Ag / M)
- Folklore (WP)
- Fortune (F)
- Heal (Int)
- Leadership (WP)
- Magic (Int / WP)
- Masonry (M)
- Orientation (Int)
- Perception (WP)
- Resilience (T)
- Ride (Ag)
- Shooting (Ag)
- Smithing (M)
- Stealth (Ag)
- Survival (WP)
- Tracking (Int)

Each skill tracks:
- **Skill Ranks**: Your training level
- **Modifiers**: Additional bonuses or penalties
- **Attribute Values**: Auto-calculated from character attributes
- **Total**: Automatic calculation of Ranks + Modifiers + Attribute
- **Dice Color**: Black, Red, White, or default
